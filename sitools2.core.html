sitools2.core.pySitools2 module¶

This is a generic python Sitools2 tool pySitools2 tool has been designed to perform all operations available within Sitools2. The code defines several classes SitoolsInstance, Field, Query, Dataset and Project. Example of application : A Solar tool to request and retrieve SDO data from IAS (Institut d’Astrophysique Spatiale) see http://sdo.ias.u-psud.fr/python/sdo_client_idoc.html

@author: Pablo ALINGERY for IAS 28-08-2012
